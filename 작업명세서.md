프로젝트명: MacLauncher Remote
1. 프로젝트 개요

iPhone을 Mac의 무선 리모트 앱 런처로 활용
연결 방식: 동일 Wi-Fi 네트워크만 지원 (Bonjour 기반 무선)

2. 배포 전략 (중요 업데이트)

- iOS 앱: App Store,정식 App Store 심사 및 배포
- macOS 앱: 웹사이트 직접 배포,App Store 미사용. GitHub Releases 또는 개인 웹사이트에서 .dmg 파일 제공

macOS 앱 배포 상세:

배포 형식: .dmg 패키지 (설치 프로그램 형태)
코드 서명: Developer ID Application Certificate로 서명 (필수)
Gatekeeper 대응: 사용자에게 "Right-click → Open" 또는 터미널 명령 안내
자동 업데이트: Sparkle 프레임워크 추천 (버전 업데이트 알림 및 자동 다운로드)
설치 위치: /Applications 폴더 추천
제공 채널: GitHub Releases + 프로젝트 웹사이트

3. macOS Companion App 요구사항

Menu Bar 앱 (Status Item)
백그라운드 실행 (LaunchAgent 지원, 로그인 시 자동 실행 옵션)
설치 가이드: iOS 앱 내에 상세 PDF 또는 웹 링크 제공
다운로드 방법
Gatekeeper 우회 방법
필요한 권한 (Full Disk Access, Accessibility)

앱 크기 최소화 (아이콘 추출 기능만 포함)

4. 연결 및 보안 (기존 유지 + 보완)

무선 Wi-Fi 전용 (Bonjour)
TLS + PIN 인증 + allowlist
연결 상태 실시간 표시 + 테스트 기능

5. iOS 앱 기능

가로/세로 완전 지원
무료: 최대 4개 앱 + AdMob (세로=하단 절반, 가로=좌측 절반)
프리미엄: 일회성 인앱 구매 (추천 가격 ₩7,900 ~ ₩9,900)
Mac 앱 다운로드 바로가기 버튼 (iOS 앱 내에 "Mac 앱 다운로드" 버튼)

6. 온보딩 및 사용자 가이드

iOS 앱 첫 실행 시:
Mac 앱 다운로드 안내
Mac 앱 설치 방법 상세 가이드
Mac 앱 실행 및 권한 설정 안내
Wi-Fi 확인 및 페어링 진행


7. 보안 및 프라이버시

macOS 앱: 최소 권한 요청
iOS 앱: Local Network 권한
Privacy Manifest 완비

8. 개발 단계 추천

Phase 1: Mac 앱 개발 + Bonjour 서비스 + 기본 앱 실행 기능
Phase 2: iOS 앱 + 네트워킹 + 페어링
Phase 3: UI (가로/세로, 광고, 프리미엄)
Phase 4: 배포 패키징 (.dmg + Sparkle) + 설치 가이드
Phase 5: 테스트 및 App Store 제출
